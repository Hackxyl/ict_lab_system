# Kiangini ICT Lab Management System

A modern Django-based ICT laboratory management system developed for the Kiangini ICT Department.

## Features

* User authentication and role management
* Student dashboard
* Officer dashboard
* Administrator dashboard
* Computer booking system
* Session management
* ICT announcements
* Laboratory management
* Responsive modern UI
* System administration services pages
* Network support services
* Help desk support

## Technologies Used

* Python
* Django
* HTML5
* CSS3
* JavaScript
* SQLite
* Chart.js
* Font Awesome
